✅ How to Use the Telemetry Analysis Tool

Upload your Telemetry (PUC) file

  • Use the upload panel to select a single file at a time.

The system processes the file automatically

  • It runs anomaly detection + summary generation.

View instant results

  • You will see classification output, detected issues, and a clear explanation of anomalies.

Download the report

  • Get a formatted summary for documentation or sharing.

  • Upload one file at a time and get results in seconds.

Clean & Intuitive Interface
